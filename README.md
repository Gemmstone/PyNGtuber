## Python Native Instalation

Requires `Python 3.11`
- PyQt6
- pyaudio
- pillow
- PyQt6-WebEngine
- beautifulsoup4
- mido
- numpy
- pynput
- dateutils
- twitchapi
- sounddevice
- pyautogui
- psutil
  
  ```py
  pip install pyqt6 pyqt6-webengine pillow pyaudio beautifulsoup4 mido numpy pynput dateutils twitchapi sounddevice pyautogui psutil
  ```
